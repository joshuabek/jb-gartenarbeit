## Deployment

### Option 1: GitHub Pages
1. Repository erstellen
2. `index.html` hochladen
3. Settings → Pages → Branch: main → Save
4. Seite ist live unter `https://benutzername.github.io/jb-gartenarbeit`

### Option 2: Strato (empfohlen)
1. Im Strato-Konto unter „Website & Shop" → Webspace öffnen
2. `index.html` per Dateimanager oder FTP hochladen
3. Direkt erreichbar unter `https://jb-gartenarbeit.de`
